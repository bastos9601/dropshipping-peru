## 🧪 Pruebas

### Prueba 1: Acceso a la página
- [ ] Iniciar sesión en el sistema
- [ ] Ver el botón "Importar ML" en el menú (verde)
- [ ] Hacer clic y verificar que carga la página
- [ ] Ver el buscador y las sugerencias

### Prueba 2: Búsqueda de productos
- [ ] Escribir "audífonos bluetooth" en el buscador
- [ ] Hacer clic en Buscar
- [ ] Esperar 3-5 segundos
- [ ] Verificar que aparecen productos
- [ ] Ver imágenes, precios y botones

### Prueba 3: Ver detalles de producto
- [ ] Hacer clic en "Importar" en cualquier producto
- [ ] Se abre el modal
- [ ] Ver imagen grande
- [ ] Ver descripción completa
- [ ] Ver precio en ML
- [ ] Ver stock disponible

### Prueba 4: Configurar margen
- [ ] Cambiar el margen de ganancia a 60%
- [ ] Verificar que el precio de venta se actualiza
- [ ] Cambiar a 100%
- [ ] Verificar nuevamente el cálculo

### Prueba 5: Importar producto
- [ ] Seleccionar una categoría (opcional)
- [ ] Hacer clic en "Importar a mi tienda"
- [ ] Ver mensaje de éxito
- [ ] El modal se cierra
- [ ] El producto desaparece de los resultados

### Prueba 6: Verificar producto importado
- [ ] Ir a "Mis Productos"
- [ ] Ver el producto importado
- [ ] Verificar que tiene el precio correcto
- [ ] Verificar que tiene la imagen
- [ ] Verificar que tiene la descripción

### Prueba 7: Ver en tienda pública
- [ ] Ir a tu tienda pública (tudominio.com/tutienda)
- [ ] Ver el producto importado
- [ ] Verificar que se ve correctamente
- [ ] Hacer clic en "Comprar por WhatsApp"
- [ ] Verificar que el mensaje incluye el producto
